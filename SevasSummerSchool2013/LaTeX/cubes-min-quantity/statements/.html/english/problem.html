Decomposition into a sum of cubes
time limit per test
1 second
memory limit per test
64 megabytes
input
stdin
output
stdout

Given a positive integer N, you should decompose it to the minimum quantity of positive perfect cubes. The program should print this minimum quantity.

Input

A single positive integer N (1 ≤ N ≤ 106).

Output

Program should print one positive integer — the minimum quantity of cubes.

Sample test(s)

Input
33
Output
5

Note

33 = 23 + 23 + 23 + 23 + 13, and 33 cannot be decomposed into a sum of less than five positive cubes.
